# 📝 Text Summarizer App

An AI-powered Text Summarization web application built using FastAPI, Hugging Face Transformers, and T5 Model.

The application allows users to input long text and generate concise summaries using Natural Language Processing (NLP).

---

## 🚀 Features

- AI-based text summarization
- FastAPI backend
- Hugging Face T5 Transformer model
- Clean web interface
- Supports long paragraph summarization

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Transformers (Hugging Face)
- PyTorch
- HTML/CSS
- Jinja2
- uvicorn

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/bhawnakushwaha/Text-summarizer.git
cd Textsummarizer
```

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv env
```

Activate environment:

#### Windows

```bash
env\Scripts\activate
```
---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

```bash
uvicorn app:app --reload
```
